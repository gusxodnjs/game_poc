# 작은정복자들 — 시나리오 v1

> **범위**: PoC v7.2 (Unity 6 LTS / iOS)
> **대상 화면**: 스플래시 → **PlanetSelect (IMGUI)** → PlayScene
> **톤 레퍼런스**: 피크민 블룸 + 스타듀밸리. 따뜻하고 작은 단어. SF 톤 금지.
> **상위 문맥**: 스플래시 카피 *"걸음마다, 자라나는 세계"* / 시작 버튼 *"첫 걸음 떼기"* 와 운율 통일.
> **핵심 메타포**: 세 곳 모두 **원시 상태**에서 시작. 산책 한 걸음마다 행성이 조금씩 푸르러진다. (스플래시의 *황폐 행성 → 푸른 지구* 연출과 일관)

## 1. 행성 3종 최종 확정

세 행성은 **모두 원시 단계의 서로 다른 형태**다. 도시·해변·풍성한 숲처럼 이미 발달한 환경은 회복의 여지가 없으므로 본 게임 컨셉에 부적합. 세 행성은 **불(잠든 화산) / 얼음(얼어붙은 평원) / 모래(메마른 들녘)** 의 가장 큰 색·온도 차로 차별화한다.

각 행성은 카드 한 장(타이틀 + 1~2문장 lore + 미니 풍경)으로 표시되고, 선택 시 색조 필터 한 장으로 PlayScene이 물든다.

> **메타포 분리**: 시작 종 풀은 "산책지에서 발견할 종"의 슬롯이지, "행성에 이미 사는 생물 목록"이 아니다. 본문 lore에는 종 이름을 직접 등장시키지 않는다.

### 1.1 `volcano` — **잠든 화산**
- **표시 이름**: 잠든 화산
- **Lore (1~2문장)**:
  검은 용암이 식어 굳고, 갈라진 틈 사이로 흰 김이 가늘게 오른다.
  바위는 아직 따뜻하고, 무언가가 자랄 자리는 비어 있다.
- **시작 종 풀 (4)**: `dandelion` / `foxtail_grass` / `white_clover` / `ladybug`
- **보너스 종 (1)**: `white_clover` — *"가끔 네 잎짜리도 발견됩니다"* 의 행운 모티프가 *식은 자리에 처음 자라는 풀* 정서와 정합.
- **색조 톤 권장**: 식은 용암 + 잿빛 적갈 `#5C3B33` (Shader Hue/Tint, 채도 -15, 명도 -5)
- **분위기 키워드**: 검은 바위, 흰 김, 잔열, 정적

### 1.2 `ice` — **얼어붙은 평원**
- **표시 이름**: 얼어붙은 평원
- **Lore (1~2문장)**:
  회청색 얼음이 지평선까지 깔리고, 바람이 한 번 길게 분다.
  깊은 정적 아래, 무언가가 풀리기를 기다리고 있다.
- **시작 종 풀 (4)**: `dandelion` / `foxtail_grass` / `honeybee` / `ladybug`
- **보너스 종 (1)**: `honeybee` — *얼음 새벽을 가장 먼저 깨는 작은 날갯짓*. 정적인 풍경 위 첫 움직임의 단서.
- **색조 톤 권장**: 새벽 회청 `#6E8AA0` (Shader Hue/Tint, 채도 -20, 명도 +3)
- **분위기 키워드**: 얼음, 정적, 회청, 긴 바람

### 1.3 `desert` — **메마른 들녘**
- **표시 이름**: 메마른 들녘
- **Lore (1~2문장)**:
  황토색 모래가 끝없이 펼쳐지고, 갈라진 땅 위로 햇빛만 내려앉는다.
  바람 한 결이 어디선가, 작은 무언가를 실어 온다.
- **시작 종 풀 (4)**: `dandelion` / `foxtail_grass` / `cherry_blossom` / `ladybug`
- **보너스 종 (1)**: `cherry_blossom` — *"봄 한때만 피는 분홍 꽃"* 의 짧고 인상적인 만남이 *바람에 실려 온 한 잎* 정서와 직결.
- **색조 톤 권장**: 황토 + 메마른 갈색 `#B89968` (Shader Hue/Tint, 채도 -10, 명도 +5)
- **분위기 키워드**: 황토, 갈라진 땅, 햇빛, 마른 바람

```
세 개의 작은 세계가, 당신의 첫 걸음을 기다립니다.

아직 아무것도 자라지 않은 곳들입니다.
어느 행성에 발을 디딜지, 천천히 골라 주세요.
```

## 3. 행성별 진입 시나리오 *(PlanetSelect → PlayScene 사이 한 컷)*

> 카드 선택 → "여기로 첫 걸음" 확정 시 1초 페이드와 함께 IMGUI 한 단락 표시. 2~4문장. 줄바꿈 보존.

### 3.1 잠든 화산 진입
```
검은 바위가 신발 끝에 닿습니다.
틈 사이로 흰 김이 한 번 오르고, 다시 가라앉습니다.
첫 걸음을, 천천히 떼어 보세요.
```

### 3.2 얼어붙은 평원 진입
```
얼음이 신발 아래에서 조용히 울립니다.
바람이 한 번 길게 지나가고, 평원은 다시 잠잠해집니다.
숨을 한 번 고르고, 한 걸음 내디뎌 보세요.
```

### 3.3 메마른 들녘 진입
```
마른 흙이 발끝에 부서집니다.
바람 한 결이 어딘가에서, 작은 무언가를 실어 옵니다.
그 결을 따라, 한 걸음 떼어 보세요.
```

## 4. 시작 버튼 카피

**결정: 공통 카피 1종 유지** ("첫 걸음 떼기"의 자산성을 살리되, 행성별 차별화는 인트로 단락이 담당).

| 위치 | 카피 |
|------|------|
| PlanetSelect 카드 하단 (행성 미선택 시) | *"이 행성 살펴보기"* |
| PlanetSelect 확정 버튼 (행성 선택 후) | **"여기로 첫 걸음"** |
| Splash → PlanetSelect 진입 버튼 (기존) | *"첫 걸음 떼기"* (변경 없음) |

> **운율 근거**: "첫 걸음 떼기" → "여기로 첫 걸음" — "걸음" 명사를 공통 앵커로 유지, 동사("떼기")는 행성 선택 행위에서 생략해 더 가볍게.
> **대안 (Open Question, 4.1)**: 행성별로 *"용암 위로 / 얼음 위로 / 모래 위로"* 분기. 채택 시 카드 정체성이 강해지나, 텍스트 자산 3배 및 운율 통일이 깨질 위험. **권장: 공통 1종**.

## 5. 영구성 정책 권고

**결정: 매 세션 자유 선택 (영구 잠금 없음)**.

| 옵션 | 장점 | 단점 | PoC 적합도 |
|------|------|------|-----------|
| A. 1회 선택 후 잠금 | 선택의 무게감, 진정성 | 시연 시 다른 행성 보여주려면 재설치 필요 | **부적합** |
| B. 매번 PlanetSelect 통과 | 시연 자유도 최고, 풍경 변주 체감 | 선택의 의미 약화 | **권장** |
| C. 마지막 선택 기억 + 우측 상단 "행성 바꾸기" | 일상 사용엔 편하고 시연도 가능 | 추가 UI 1포인트 | 차선 |

**근거**: PoC는 *체험 시연*이 목적. 행성 3개 풍경을 5분 내에 모두 보여줘야 하므로 **B**가 적합. 출시 단계에서는 **C** 또는 **A**로 승격 검토.

**구현 메모 (PM 참고)**:
- 직전 선택 행성 ID는 `PlayerPrefs("last_planet", "volcano")` 로만 저장 → PlanetSelect 진입 시 기본 포커스 카드로 사용. 잠금은 걸지 않음.

## 6. 시작 종 풀 검증 메모

각 행성 4종 + 보너스 1종 = 5종 등장. 6종 전체 중 부분집합 4종 + 보너스 1종 권장 충족.

**원칙**: 원시 환경엔 *가장 강한 정착자*부터 들어온다. 잡초류(민들레/강아지풀)와 익충(무당벌레)을 3행성 공통 안전망으로 깔고, 행성별 시그니처 1종으로 회복 단서를 부여한다.

| 행성 | 종 구성 | 톤 정합성 근거 |
|------|---------|---------------|
| 잠든 화산 | 민들레 / 강아지풀 / 토끼풀 / 무당벌레 (+ 보너스 토끼풀) | 식은 용암 자리에 가장 먼저 자리잡는 강한 잡초류 + 익충. 토끼풀의 "네 잎" 행운 모티프가 *재가 식은 자리에 처음 자라는 풀* 정서와 결합. 꿀벌·벚꽃은 잔열·정적 톤과 약하게 정합되어 제외. |
| 얼어붙은 평원 | 민들레 / 강아지풀 / 꿀벌 / 무당벌레 (+ 보너스 꿀벌) | 잡초류 2종 + 정적인 풍경 위 첫 움직임으로서의 꿀벌 + 익충. 보너스 꿀벌이 *얼음 새벽을 가장 먼저 깨는 작은 날갯짓* 단서로 작동. 토끼풀·벚꽃은 얼음 정서와 약함. |
| 메마른 들녘 | 민들레 / 강아지풀 / 벚꽃 / 무당벌레 (+ 보너스 벚꽃) | 잡초류 2종 + 바람 타고 온 벚꽃 한 잎 + 익충. 벚꽃은 *마른 바람에 실려 온 작은 무언가* 라는 lore와 직결. 보너스 벚꽃의 "짧은 한때" 희소성이 메마른 풍경 위 한 컷과 정합. 꿀벌·토끼풀은 황토 톤과 약함. |

**공통 등장**: 민들레 + 강아지풀 + 무당벌레 (3행성 모두). *어디서나 가장 먼저 자리잡을 수 있는 종*이라는 원시 환경 정착 원칙과 일치, 초보자 안전망 역할.
**행성 시그니처**: 토끼풀(화산) / 꿀벌(얼음) / 벚꽃(사막) — 보너스로 강조.
**기존 산림(forest) 시그니처였던 토끼풀의 이동**: 풍성한 숲이 사라졌으므로, *식은 자리에 처음 자라는 풀* 이미지가 가장 강한 잠든 화산으로 이동.

## 7. 후속 작업 인계 (Open Questions / 메인 스레드용)

1. **PNG 카드 시안** (pixellab 라운드): 행성 3종 1024px 마스터 + 256px 카드 썸네일. 위 색조 톤(`#5C3B33` / `#6E8AA0` / `#B89968`)을 카드 베이스 컬러로 사용 권장. **공통 가이드**: 도시 건물/모래사장 야자수/풍성한 숲 금지. 회색·검은·흰·적갈·황토의 황량함이 기본 톤.
2. **Unity PlanetSelect 데이터** (unity 라운드): `data/planets.json` 신규 — `planet_id` / `display_name` / `lore` / `tint_hex` / `species_pool[4]` / `bonus_species`. 본 문서 §1 그대로 매핑.
3. **Shader 색조 필터**: 한 장의 Unlit/Tint 머티리얼에 행성별 `_TintColor` 만 주입. (PoC: 카메라 PostProcess 대신 스프라이트 머티리얼 단일화 권장 — iOS 부하 절감).
4. **잔여 결정 필요**:
   - (4.1) 시작 버튼 공통 vs 분기 — 본 문서 §4 권장 **공통**. PM 확정 요청.
   - (7.1) 행성 카드에 시작 종 풀 아이콘 4개 미리보기를 노출할지 여부 — PoC 정보량/시각 복잡도 트레이드오프. **권장: 보너스 종 1개만 큼지막히 노출** (선택의 단서는 주되, 학습 부담은 줄임).
   - (7.2) 회복 진행도(걸음 수 누적)에 따라 행성 색조가 점진적으로 푸르러지는 연출 — PoC v7.2 스코프 외. **권장: 다음 마일스톤(라이브 회복 연출)으로 이관**, 본 PoC는 색조 1장 고정.

## 8. 자기 검증 체크리스트

- [x] 개발자가 추가 질문 없이 PlanetSelect IMGUI 화면을 구현할 수 있는가? → §1 표 + §2/§3 텍스트 + §4 버튼 카피로 충분.
- [x] 아티스트가 카드 시안을 그릴 수 있는가? → §1의 색조 톤 + 분위기 키워드로 충분. §7.1에 *발달된 환경 금지* 가드 명시.
- [x] 5분 내 읽을 분량인가? → 본문 약 1.5p, 텍스트 자산 단락 7개 (각 2~4문장).
- [x] 톤이 SF로 새지 않는가? → 단어 검수: "행성"은 환경 테마, "우주/SF/스테이션/탐사대" 어휘 0회. 표시 이름에서 "행성" 어휘 제거.
- [x] 기존 스플래시 자산과 충돌하지 않는가? → "첫 걸음", "걸음" 앵커 유지, "떼기" 동사 1회 등장 유지.
- [x] **모든 행성이 회복할 여지가 있을 만큼 원시 상태인가?** → 잠든 화산(식은 용암, 빈 자리) / 얼어붙은 평원(정적, 풀림 대기) / 메마른 들녘(빈 모래, 바람 한 결) — 세 곳 모두 *아직 아무것도 자라지 않음* 명시.
- [x] **종 풀이 "행성 거주 생물 목록"으로 오독될 여지가 없는가?** → §1 메타포 분리 단서 + lore에 종 이름 미등장으로 분리 유지.
