모든 텍스트 자산(카피, 시나리오, 버튼)은 따뜻하고 작은 단어로 통일. 시적이되 모호하지 않게. 한 단락은 2~4문장.

**Why:** 사용자가 "기존 스플래시 브랜드 톤 유지", "피크민 블룸 + 스타듀밸리 계열, SF 톤 X" 명시. 또한 "뭘 말하는지 모르겠어" 발언에서 보듯 압축적·명료한 한국어를 선호. 모호한 시적 표현은 역효과.

**How to apply:**
- 어휘 금지: "우주, SF, 탐사대, 스테이션, 미션, 시스템" 등 기계·우주선 계열.
- 어휘 권장: "걸음, 발자국, 결, 한 컷, 햇볕, 바람, 안개, 이끼" 등 신체·자연·시간 단위.
- 앵커 단어 **"걸음"** 은 모든 화면 카피에 최소 1회 등장 (스플래시 *"걸음마다, 자라나는 세계"*, 시작 *"첫 걸음 떼기"*, PlanetSelect 확정 *"여기로 첫 걸음"*).
- 행성 이름은 4~6자 한글, 자연 명사 + 형용사 조합 (예: 푸른 숲 / 틈새 도시 / 바람 해변).
- 새 카피 작성 시 위 두 자산과 운율을 맞춰 검증.
